# Allien

**Allien** is a fun and interactive website for kids and learners!

## 🚀 What You Can Do in Allien

### 🎮 Games Library
Play a growing collection of fun games, such as:
- Memory match
- Tic-tac-toe
- (and more you can add!)

### 🧠 Quizzes
Test what you know with interactive quizzes:
- Multiple-choice questions
- Instant feedback
- Track your score

### 🤖 Own Allien AI Assistant
Ask questions or get help with:
- Learning new concepts
- Getting hints for games and quizzes
- Exploring fun topics

### 🔍 Own Allien Search
Quickly find what you want in the app:
- Search for games
- Search for quizzes
- Search for help topics or categories

---

## 🧩 Project Goals

- Make learning fun and easy to use
- Play games and learn new things
- Use a smart AI helper

---

## 🛠️ Tech Stack

- **Frontend:** React (TypeScript)
- **Build:** Vite

---

## ▶️ How to Start

1. Install [Node.js and npm](https://nodejs.org/)
2. Clone this repo: `git clone https://github.com/marcusz0608/Allien.git`
3. Go into the repo folder: `cd Allien`
4. Install dependencies: `npm install`
5. Start the app: `npm run dev`
